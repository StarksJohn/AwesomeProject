# AwesomeProject
Scaffolding for react native project

Install:
    git clone --recurse-submodules https://github.com/StarksJohn/AwesomeProject.git 
    
Update:

    如果主项目&&子模块都有更新,执行: git pull --recurse-submodules

    如果只更新子模块,执行: git submodule update --remote
